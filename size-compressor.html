Tips for best results

💡

For images: Use JPEG or WebP format. 75-85% quality is visually identical to the original but 50-70% smaller in file size.

📄

For PDFs: "Medium" compression gives a good balance. "High" is best for scanned documents. Text-heavy PDFs may see smaller reductions since text is already compact.

🔒

100% private: All compression runs inside your browser. Your files are never sent to any server.

Frequently Asked Questions

How do I compress a PDF without losing quality? +
Use the "Low" compression level — it re-renders pages at a slightly lower resolution with 88% JPEG quality. For most documents, the visual difference is unnoticeable while the file size drops by 20-40%. Avoid "High" for documents you'll print or share professionally.
How much can I reduce a PDF file size? +
It depends on the content. Image-heavy PDFs (scanned documents, brochures) typically compress by 50-75%. Text-heavy PDFs compress less (10-30%) since text is already stored efficiently. Use "High" compression for the maximum reduction.
Can I compress multiple images at once? +
The Size Compressor handles one file at a time. If you need to convert between formats in bulk, use the File Converter tool which supports multiple images and packages them as a ZIP download.
Are my files private? +
Yes. All compression happens inside your browser using JavaScript. Your images and PDFs are never sent to any server. Nothing is stored after you close the tab.
